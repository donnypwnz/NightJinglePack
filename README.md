Night 3DS Jingle Pack - Beta

This repository contains a custom Nintendo 3DS jingle pack intended for use with compatible frontends and emulation setups. The pack maps game titles to audio files using a JSON configuration, allowing a short jingle to play when a game is selected or launched.

The structure is simple and easy to modify. Each entry in the configuration links a game name to an audio file and uses regex matching to ensure flexibility across different naming formats. The audio files are organized in a clear folder structure to keep everything maintainable and scalable.

This project is designed for personal use and customization. You are free to expand the pack by adding new entries, modifying regex patterns, or replacing audio files to better suit your own setup.

To use the pack, place the files in the appropriate directory for your frontend or emulator and ensure that JSON-based jingle packs are supported. The system will automatically match game titles to the corresponding audio files based on the defined rules.
